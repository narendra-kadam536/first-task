# Personal Introduction Program 

This is a simple Python beginner project that asks the user for their
**name**, **age**, and **hobby**, then displays a friendly welcome message.

---

##  Project Files

- personal_intro.py → Main Python program
- requirements.txt → Required Python packages (none needed)
- README.md → Project description and instructions

---

## How to Run the Program

1. Make sure Python is installed  
   Check using:
   bash
   python --version
